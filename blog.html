Var

  • Can be reassigned and redeclared
  • Can be declared without initialization
  • Global or function scoped

Let

  • Can be reassigned and cannot be redeclared
  • Can be declared without initialization
  • Block scoped

Const

  • Neither can be reassigned and nor can be redeclared
  • Must be initialized at the time of declaration
  • Block scoped

map: takes each element of the array, runs the described operation with each element and puts each element in a new array and finally returns the new array forEach: takes each element of the array, runs the described operations with each element but doesn't return anything like map. filter: takes each element of the array, checks the condition given in the function for each element. Puts the elements that satisfies the condition in a new array and finally returns that new array. find: similar to filter. just returns the first element that fulfills the condition instead of returning an array of elements.

A semantic tag is something that clearly describes its meaning to both the browser and the developer. Some semantic tags are header, main, footer, aside, article etc. These tags are introduced in HTML5 for making html files more meaningful for both humans and machines.
